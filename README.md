# Flatten

## How to Use

```javascript
const flatten = require('@syednashikaman/flatten');

const nestedArray = [1, [2, 3, [4, 5, [6, 7]]]];
flatten(nestedArray); // [1, 2, 3, 4, 5, 6, 7]
```

## How to Install

```sh
$ npm install --save @syednashikaman/flatten
```

or

```sh
$ yarn add @syednashikaman/flatten
```sh
